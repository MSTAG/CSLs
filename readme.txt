This is a .CSL (http://en.wikipedia.org/wiki/Citation_Style_Language) that we use to convert entries in Zotero to a Dspace-import-ready .csv. 

The proper text prefix to put before the clipboard output is:
id,collection,dc.identifier.citation[en_US],dc.contributor.author[en_US],dc.contributor.corporatename[en_US],dc.contributor.deptlab[en_US],dc.contributor.id[en_US],dc.contributor.meetingname[en_US],dc.contributor.sponsor[en_US],dc.date.issued,dc.description[en_US],dc.description.abstract[en_US],dc.description.version[en_US],dc.language[en_US],dc.relation.ispartof[en_US],dc.relation.ispartofcommunity[en_US],dc.relation.uri,dc.rights[en_US],dc.rights.uri,dc.publisher[en_US],dc.subject[en_US],dc.title.alternative[en_US],dc.title[en_US],dc.type[en_US]



After that, paste the clipped bibliography from Zotero on the second line.  Then save as .csv.
